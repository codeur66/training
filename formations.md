# Formations Consultants - Axes de Compétences

**💻 1. [Développeur][Python][Microservices]**
Maîtriser l'architecture microservices avec Python : conception de services découplés, communication inter-services (REST, gRPC, message queues), gestion des transactions distribuées et patterns de résilience.
